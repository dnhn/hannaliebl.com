## Hanna Liebl's homepage!

Built with [Hugo](https://gohugo.io/).

---
1. Clone repo
2. [Install Hugo](https://github.com/spf13/hugo#choose-how-to-install)
3. cd into project folder
4. Run `hugo server` to serve the site at `localhost:1313``
